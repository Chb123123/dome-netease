# dome-netease

## 描述

- 主要用于播放音乐，以及播放MV
- 当前仓库的代码为打包后的代码
- 项目源码：https://github.com/Chb123123/demo/tree/branch/%E7%BD%91%E6%98%93%E4%BA%91%E9%9F%B3%E4%B9%90/demo-netease

## API地址

https://neteasecloudmusicapi.vercel.app/#/?id=neteasecloudmusicapi

## 缺陷

- 由于接口做了缓存处理，重复调用相同的接口可能会请求不到数据
- 由于API提供的音乐URL并不多，所以可以听的歌曲并不多

## 项目在线查看
https://chb123123.github.io/dome-netease/#/

## 技术栈
Vue-cli + Vant + Vue-router + axios
